# 数据结构与算法

## 排序算法
| 特性    | 快速排序        | 归并排序         | 堆排序         |
| ----- | ----------- | ------------ | ----------- |
| 最优复杂度 | O(n log n)  | O(n log n)   | O(n log n)  |
| 最坏复杂度 | O(n²)       | O(n log n)   | O(n log n)  |
| 空间    | O(log n)    | O(n)         | O(1)        |
| 稳定性   | 不稳定         | 稳定           | 不稳定         |
| 特点    | 原地排序快，最坏需优化 | 稳定，适合链表或外部排序 | 空间小，适合对空间敏感 |

## 多叉树

| 结构             | 数据存储 | 叶子节点 | 是否链表 | 查询路径   | 插入策略      | 特点      |
| -------------- | ---- | ---- | ---- | ------ | --------- | ------- |
| **B树（B-Tree）** | 所有节点 | 不一定  | ❌    | 可能中途结束 | 满就分裂      | 基础结构    |
| **B+树**        | 只在叶子 | ✅ 全部 | ✅    | 一定到叶子  | 满就分裂      | 范围查询强   |
| **B*树**        | 只在叶子 | ✅ 全部 | ✅    | 一定到叶子  | **先借再分裂** | 空间利用率更高 |
