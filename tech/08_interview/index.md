## 系统设计

### 单系统设计

- **JVM**：一个类的加载卸载过程
  - 类加载器机制
  - 方法区、堆、栈、GC 原理
- **Spring**：一个 Bean 的装载卸载过程
  - IOC / DI 原理
  - AOP 代理和增强处理
- **MySQL**：一条 SQL 执行过程
  - SQL 解析 → 优化 → 执行 → 存储引擎
  - 索引 / 锁 / 事务 / 隔离级别
- **MyBatis**：一条 SQL 执行过程
  - SQL 解析 → 参数映射 → 缓存机制（一级 / 二级缓存）
- **Kafka**：一条消息发送到消费的过程
  - 分区 / ISR / ACK / 消息确认机制
- **Netty**：一次 IO 的完整过程
  - 0Copy / EventLoop / ChannelPipeline / ByteBuf
- **ElasticSearch**：一次 ES 查询过程
  - 索引 / 分片 / 倒排索引 / 查询分布式执行

---

### 单系统局部集群化

#### 服务端集群化

- **RPC**：一次 RPC 请求过程
  - 序列化 / 反序列化
  - 负载均衡 / 心跳 / 容错机制
- **Zookeeper**：集群初始化与重选举过程
  - 节点注册 / Leader 选举 / 集群健康检查

#### 三方中间件集群化

- **Redis**：集群初始化与重选举
  - 主从复制 / 哨兵 / 故障转移
- **MySQL**：集群初始化与重选举
  - 主从复制 / GTID / 故障切换
- **Kafka**：集群初始化与重选举
  - 分区副本 / ISR / Leader 选举

### 分布式系统设计

- 负载均衡
  - 轮询 / 权重 / 哈希 / 一致性哈希
